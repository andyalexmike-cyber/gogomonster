# Go-Go! Duck Roulette

[🇰🇷 한국어 (Korean)](README.md)

![Go-Go! Duck Roulette](thumb.png)

A random ranking game featuring racing ducks!

[Play Game](https://hyojun.me/gogoduck)

## Development & Build

This project is built using Webpack.

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```
You can view the app at `http://localhost:9000`.

### Production Build

```bash
npm run build
```
The build artifacts will be generated in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
Run the production build locally to verify it works as expected.
